Function: Apply_Parenthesis_Rule (input text)
    copy input text into new variable output text
    for every character in the output text:
        if it is a parenthesis:
            check if it has a space before it and after it
            if it does not have a space before it:
                add a space before it
            if it does not have a space after it:
                add a space after it
    return output text
